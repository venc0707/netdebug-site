О сервисе

NetDebug — это SaaS-инструмент для тестирования API и сетевых сценариев. Сервис предоставляет доступ к тестовой инфраструктуре через HTTP API.

Сервис предоставляет инструменты для тестирования и управления сетевыми соединениями. Функциональность сервиса может включать использование защищённых каналов передачи данных.

Возможности

  • — Тестирование отказоустойчивости API
  • — Проверка обработки ошибок
  • — Мониторинг доступности сервисов
  • — Интеграционное тестирование

Как это работает

1. Регистрация
2. Тестовый API-ключ (24 часа)
3. Выполнение запросов
4. Оплата доступа

Пример

curl https://eu-1.netdebug.dev/health

curl -H "X-Debug-Region: EU-1" https://api.your-service.com

Оплата и доступ

Оплата осуществляется через ЮKassa. После успешной оплаты доступ активируется автоматически. Доступ предоставляется в личном кабинете.

Возврат средств

Возврат возможен в течение 3 дней при условии, что API-ключ не использовался. После начала использования услуга считается оказанной.
